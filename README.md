BIN是一套前端JS框架，目的是提供一套可用于移动应用、WEB应用和桌面应用的统一框架。<br/>

## 目录说明
* bin : 框架代码 <br/>
* config : 配置模板<br/>
* index-spa.html : 单页应用引导文件<br/>
* index-web.html : WEB应用页面文件<br/>
